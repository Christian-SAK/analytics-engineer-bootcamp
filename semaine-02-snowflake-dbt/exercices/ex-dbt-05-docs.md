# ex-dbt-05 — Documentation métier

## Énoncé

1. Ajoutez des descriptions métier (français) à tous les modèles marts dans `schema.yml`.
2. Générez `dbt docs` et exportez une capture du **lineage** (DAG).
3. Rédigez un paragraphe expliquant le **grain** de `fct_order_lines` pour un stakeholder non technique.

## Validation

- Descriptions visibles dans `dbt docs serve`
- Paragraphe dans `exercices/mes-notes-dbt.md`

## Livrable

`schema.yml` enrichi + note markdown.
